Sunglasses Hub Dataflow

This pipeline extracts data from the Sunglasseshut web page that will be synced into a Power BI dashboard.

The tools used for this project are the following:
  • > Azure for hosting the infrastructure.
  • > Terraform as IaaC for infrastructure provisioning.
  • > Airflow for orchestration.
  • > Docker for containerizing the pipeline.
  • > Insomnia for obtaining the code that sends requests to the Web page API.
  • > Power BI for data visualization.
  • > Python as the main programming language.
Project's Architecture
Steps
  • > Scraping the data using Insomnia and Python.
  • > The extracted data is converted into a DataFrame that is uploaded to an Azure Storage account using the Azure Identity and Azure Blob Storage client libraries.
  • > The data is cleaned and the data types are validated using Pydantic's data classes.
  • > Finally, the data is delivered to Azure Synapse (Data warehouse) and Azure Database for PostgreSQL - Flexible Server.
  • > Users can now analyze the data using Power BI or any other visualization tool they prefer.

Dunes of Influence

Quantitative Prediction of Influential Individuals' Influence on Community Dynamics through Advanced Language Models.

This project develops a deep learning framework to estimate potential engagement metrics such as likes, comments, and retweets for new tweets, leveraging data from social media platforms like Twitter and Reddit.

The tools and techniques used for this project are the following:
  • > Data Collection: Collect data from Twitter and Reddit using their respective APIs.
  • > Pre-processing: Use techniques like sentiment analysis, topic classification, and content embeddings to extract relevant features.
  • > Modeling: Use transformer-based neural networks to predict future engagement metrics.
  • > Languages: Python for scripting and data processing.
  • > Frameworks: TensorFlow, PyTorch for building and training the models.
  • > Tools: Jupyter Notebook for experimentation and visualization, Pandas for data manipulation, Scikit-learn for additional data processing and model evaluation.
Project's Architecture
Steps
  • > Collect data from Twitter and Reddit.
  • > Pre-process the data using sentiment analysis, topic classification, and content embeddings.
  • > Feed the processed data into a transformer-based neural network.
  • > Train the model using historical data.
  • > Validate the model using various performance metrics.
  • > Predict engagement metrics such as likes, comments, and retweets for new tweets.
Results and Conclusion
Model Performance

The model demonstrated high accuracy in predicting engagement metrics, with transformer-based models outperforming traditional models. The results indicated that semantic embeddings and sentiment analysis significantly improve prediction performance. The use of multi-modal data and advanced NLP techniques enabled a deeper understanding of social media dynamics.

Evaluation of Different Models

Validation results on different models show that the Transformer and Compressed Transformer models significantly outperform the traditional MLP and LSTM models in terms of prediction accuracy.

Evaluation of Different Loss Functions

Validation results on different loss functions reveal that Mean Absolute Error (MAE) provided the best balance between precision and robustness, particularly in predicting retweets.

Overall, the project showcased the effectiveness of integrating advanced machine learning techniques for predicting social media engagement, providing valuable insights for marketers and content creators.
